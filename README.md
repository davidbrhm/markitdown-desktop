# MarkItDown Desktop

A cross-platform desktop application built with C# and Avalonia UI, using Microsoft's MarkItDown to convert files into Markdown.

## About

MarkItDown Desktop provides a modern graphical user interface for Microsoft's MarkItDown tool.

The goal of this project is to make document conversion more accessible through a desktop experience while adding features focused on usability, visualization, and local workflow management.

## Key Features

- **File & URL Conversion:** Convert supported files and URLs into Markdown.
- **Modern Desktop Interface:** A user-friendly GUI built with Avalonia UI.
- **Local Processing:** Designed with a local-first approach.

## Tech Stack

- **Language:** C#
- **Framework:** .NET 10
- **UI:** Avalonia UI (MVVM Architecture)
